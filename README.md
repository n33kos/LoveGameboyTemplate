# Love Gameboy Template

This is a simple template meant to be used as a starting point for games in the style of the original gameboy.

 - Resolution: 160 x 144
 - Default Scaling: 4x
 - Colors:
 	- Dark: (15, 56, 15)
 	- Medium: (48, 98, 48)
 	- Light: (139, 172, 15)
 	- Very Light (155, 188, 15)
